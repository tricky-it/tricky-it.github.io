ດາວໂຫລດ ແລະ ຕິດຕັ້ງ font Phetsarath OT ສຳລັບ Windows

Font Phetsarath ແມ່ນຟອນທີຈຳເປັນຕ້ອງໄດ້ມີເພາະວ່າເປັນຟອນທີ່ໃຊ້ສຳລັບພິມເອກະສານ,ລາຍງານ,ວຽກບ້ານໃນ Microsoft word, ນຳສະເໜີວຽກຜ່ານ Powerpoint ແລະອື່ນໆ ອີກຫຼາຍຢ່າງ.

ແຕ່ຖ້າຄອມເຮົາບໍ່ມີຊິເຮັດແນວໃດ, ແອັດມິນເອງກໍເຄີຍພົບບັນຫານີ້ ເພາະວ່າໄປຊື້ຄອມຢູ່ໄທແລ້ວເຂົາບໍ່ໄດ້ຕິດຕັ້ງມາໃຫ້ T___T ຖ້າພົບບັນຫານີ້ ວິທີແກ້ໄຂມີຫຼາຍວິທີ ເຊັ່ນ: ເອົາໄປຮ້ານ(ເສຍເງິນອີກແລ້ວ), ໃຫ້ໝູ່ແປງໃຫ້ (ເຂົາຊິວ່າງມາແປງໃຫ້ມື້ໃດນິ) ແລະ ວິທີສຸດທ້າຍໂຫລດມາລົງເອງ.

ກ່ອນຈະເຂົ້າຂັ້ນຕອນການຕິດຕັ້ງ ເຮົາຈະຮູ້ໄດ້ຈັ່ງໃດວ່າຄອມພິວເຕີ້ຂອງເຮົາມີແລ້ວຫຼືວ່າບໍ່ມີ ວິທີທີ່ຈະເຮັດໃຫ້ເຮັດໃຫ້ເຮົາຮູ້ຄືຕ້ອງໄປກວດສອບເບິງ. ໂດຍເປີດໂປຣແກຣມ Microsoft office(Word, Excel, Powerpoint) ອັນໃດກໍໄດ້ສຳລັບແອັດມິນຂໍເລືອກເປັນ Microsoft Word 2013.

ເມືອເປີດຂື້ນມາແລ້ວກົດໃສ່ combo box ສຳລັບປຽນຟອນດັ່ງຮູບ

ສຳລັບທີກວດສອບແມ່ນມີຫລາຍວິທີເນາະແຕ່ວ່າແອັດມິນແນະນຳວິທີນີ້ເພາະມັນງ່າຍດິ…ໂອເຄຕໍ່ໄປແມ່ນເຮົາຈະເລີມດາວໂຫລດແລ້ວຕິດຕັ້ງ. ຂັ້ນຕອນດາວໂຫລດ

  1. ຄລິກເພື່ອເລີມດາວໂຫລດ

  2. ເມື່ອດາວໂຫລດແລ້ວໆ ໄປທີ This PC (ສຳລັບ windows 10 ແລະ My computer ສຳລັບ windows 7)

  3. ກົດເປີດ File ຂື້ນມາແລ້ວດືງ file ອອກມາວາງໄວ້ນອກໜ້າຕ່າງໂປຣແກຣມ

  4. Double click file ແລ້ວກົດ install ແລ້ວລໍຖ້າ

ພຽງເທົ່ານີ້ເຮົາກໍ່ຟອນລາວງາມໆມາຕິດຕັ້ງໃນເຄືອງຂອງທ່ານແລ້ວຖ້າບົດຄວາມນີີ້ມີປະໂຫຍດ ແມ່ນກົດ Recommand ກົດ Share ເພື່ອເປັນກຳລັງໃຈໃຫ້ພວກເຮົາແນ່ເດີ
